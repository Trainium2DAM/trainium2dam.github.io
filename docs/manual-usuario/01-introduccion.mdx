# Introducción a Trainium

Trainium es una aplicación Android para gestionar tu actividad en el gimnasio desde el móvil. Con ella puedes reservar máquinas, registrar tu evolución de peso, consultar recetas y planes de nutrición, y contratar una suscripción Premium con acceso a todas las funcionalidades.

## ¿Qué puedes hacer con Trainium?

| Sección | Para qué sirve |
|---|---|
| **Máquinas** | Consultar el catálogo de equipamiento y reservar franjas horarias |
| **Mis Reservas** | Ver, filtrar y cancelar tus reservas; exportarlas al calendario del teléfono |
| **Platos** | Explorar recetas, ver calorías y tiempo de preparación, y sugerir las tuyas |
| **Registro de peso** | Apuntar tu peso diario y visualizar tu evolución en un gráfico |
| **Mi Perfil** | Editar tus datos personales, foto y configurar notificaciones |
| **Premium** | Contratar una suscripción mensual, semestral o anual |

## Requisitos para usar la app

- Teléfono Android con versión 7.0 (Nougat) o superior.
- Conexión a internet activa. Todas las operaciones de datos (reservas, perfil, platos, peso) requieren conectividad, ya que la información se almacena en la nube.
