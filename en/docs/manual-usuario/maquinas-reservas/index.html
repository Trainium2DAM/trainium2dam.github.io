    Machines and Bookings

    The machines section lets you browse the equipment available in the gym and book the machine you want to use for a specific time slot.

    Viewing the Machine Catalog​

    Tap "Machines" on the main screen to see the full equipment list. While data is loading you'll see animated gray cards (skeleton loading) that are replaced by the actual cards as soon as the information arrives.

    Each machine card shows:

    • Name of the machine.
    • Exercise category (Cardio, Back, Legs, Chest, etc.).
    • Description of the equipment.
    • Current status, indicated by a colored badge:
    BadgeColorMeaning
    AvailableGreenThe machine is operational and you can book it
    MaintenanceOrangeTemporarily out of service, cannot be booked

    When a machine is in maintenance, the card also shows the start and end dates of the period, so you know when it will be available again.

    Making a Booking​

    To book a machine it must be in Available status. Tap on its card to start the process:

    1. Select the Date​

    A calendar will open where you can choose the day you want to use the machine.

    2. Select the Start Time​

    After choosing the date, a time picker will appear. Select the time you want to start.

    3. Adjust the End Time (optional)​

    The app automatically calculates the end time by adding 30 minutes to the start time. If needed you can adjust this time, but keep in mind that the maximum allowed per booking is 1 hour. If you try to book for longer the app will warn you with an error message.

    4. Automatic Availability Check​

    Before saving the booking, the app checks two conditions:

    • That the machine is not already booked by another user in that time slot.
    • That you don't have another booking in the same time slot.

    If there's a conflict you'll see a message explaining the reason and you can choose another date or time.

    5. Booking Confirmed​

    If there are no conflicts, the booking is saved and a reminder is automatically scheduled to notify you before the start time. The reminder lead time is whatever you configured in My Profile (30 minutes by default).

    Información

    It is not possible to have two bookings in the same time slot even if they are on different machines. Plan your sessions carefully to avoid conflicts.

    Machines in Maintenance​

    When the administrator puts a machine into maintenance for a given period, the following happens:

    • The machine appears with the orange "Maintenance" badge and the booking button is disabled.
    • If you had a booking during that period, it is automatically canceled.
    • When you open the app you'll see a notice indicating the affected machines and bookings.

    Once the maintenance period ends, the machine returns to Available status and you can book it again.

    Booking Restrictions​

    RestrictionDetail
    Maximum duration1 hour per booking
    Time overlapYou cannot have two bookings at the same time
    Machines in maintenanceCannot be booked during the maintenance period
    Internet connectionRequired to check availability and confirm bookings
