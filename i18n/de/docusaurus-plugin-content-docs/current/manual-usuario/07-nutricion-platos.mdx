# Ernährung und Gerichte

Die Rubrik **Gerichte** ist der Katalog des Fitnessstudios mit Rezepten und Ernährungsvorschlägen. Jeder Benutzer kann die verfügbaren Gerichte erkunden und eigene Rezepte vorschlagen, die dem Katalog hinzugefügt werden sollen.

---

## Entdecken Sie den Katalog

Drücken Sie auf dem Hauptbildschirm **„Gerichte“**, um auf den Katalog zuzugreifen. Die Gerichte werden einzeln in einem großen Kartenformat angezeigt, ähnlich einem linearen Karussell. Verwenden Sie die **Navigationspfeile**, um zum nächsten Gericht zu wechseln oder zum vorherigen zurückzukehren.

Im Katalog erscheinen nur Gerichte, die **von einem Administrator genehmigt** wurden. Vorschläge, deren Überprüfung noch aussteht, sind für Standardbenutzer nicht sichtbar.

---

## Informationen zu jedem Gericht

Auf jeder Gerichtkarte ist Folgendes zu sehen:

| Daten | Beschreibung | 
|---|---| 
| **Foto** | Bild des Gerichts (falls beim Hochladen hinzugefügt) | 
| **Name** | Name des Gerichts oder Rezepts | 
| **Kalorien** | Ungefährer Kaloriengehalt pro Portion | 
| **Vorbereitungszeit** | Geschätzte Zeit (zum Beispiel „20 Minuten“) | 
| **Beschreibung / Rezept** | Zutaten, Zubereitungsschritte oder andere relevante Informationen |

---

## Schlagen Sie ein neues Gericht vor

Wenn Sie ein Rezept zur Aufnahme in den Katalog vorschlagen möchten, klicken Sie auf die Vorschlagsschaltfläche (Plus-Symbol o.ä.) und füllen Sie das Formular aus:

### Formularfelder

| Feld | Obligatorisch | Beschreibung | 
|---|---|---| 
| **Name des Gerichts** | Ja | Der Titel, mit dem es im Katalog erscheint | 
| **Beschreibung / Rezept** | Nein | Zutaten, Schritte, Zubereitungstipps... | 
| **Vorbereitungszeit** | Nein | Schreiben Sie die Zeit im Freitext, zum Beispiel „30 Minuten“ | 
| **Kalorien** | Nein | Ungefähre Kalorien pro Portion | 
| **Foto** | Nein | Wählen Sie ein Bild aus Ihrer Galerie |

Wenn Sie das Formular ausgefüllt haben, klicken Sie auf „Vorschlag senden“**.

### Was passiert als nächstes?

1. Ihr Vorschlag wird mit dem Status **Überprüfung ausstehend** registriert. 
2. Der Fitnessstudio-Manager überprüft sie. 
3. Wenn Sie es genehmigen, erscheint das Gericht im Katalog, sichtbar für alle Benutzer. 
4. Wenn Sie es ablehnen, wird das Gericht gelöscht, ohne dass Sie eine Benachrichtigung erhalten (wenden Sie sich an den Administrator, wenn Sie den Grund erfahren möchten).

:::Info
Sie müssen kein Premium-Benutzer sein, um Gerichte vorschlagen zu können. Diese Funktion steht allen registrierten Benutzern zur Verfügung. 
:::

---

## Gute Praktiken beim Vorschlagen von Gerichten

Um Ihrem Vorschlag eine bessere Chance auf Genehmigung zu geben:

- Geben Sie einen **klaren und beschreibenden Namen** ein, der das Gericht eindeutig identifiziert. 
- Enthält ein **detailliertes Rezept** mit Zutaten und Schritten. Je vollständiger es ist, desto nützlicher ist es für andere Benutzer. 
- Fügen Sie ein **Qualitätsfoto** des fertigen Gerichts hinzu. Vorschläge mit Fotos haben im Katalog eine größere visuelle Wirkung. 
- Geben Sie die **Kalorien** an, wenn Sie diese kennen, auch wenn es sich um ungefähre Werte handelt. 
- Stellen Sie sicher, dass das Gericht **für den sportlichen Kontext geeignet** ist: ausgewogene Rezepte, Proteinquellen, hochwertige Kohlenhydrate usw.
