# Reservierungsverwaltung

Im Bereich **„Meine Reservierungen“** können Sie alle Ihre Reservierungen einsehen, sie nach Datum oder Status filtern, nicht mehr benötigte Reservierungen stornieren und sie in den Kalender Ihres Telefons exportieren.

---

## Sehen Sie sich Ihre Reservierungen an

Drücken Sie im Hauptbildschirm auf **„Meine Reservierungen“**. Alle Ihre Reservierungen werden nach Datum sortiert geladen (das nächstliegende zuerst). Während die Liste geladen wird, werden Sie die Auswirkung des Ladens des Skeletts sehen.

Auf jeder Reservierungskarte steht:

- Name der reservierten **Maschine**. 
- **Datum** der Reservierung. 
- **Start- und Endzeit**. 
- Status (aktiv oder vergangen).

---

## Reservierungen filtern

Oben auf dem Bildschirm finden Sie die verfügbaren Filter. Sie können sie horizontal verschieben, wenn sie nicht alle auf den Bildschirm passen:

| Filter | Was zeigt | 
|---|---| 
| **Kommend** | Reservierungen, deren Endzeit noch nicht erreicht ist. Standardansicht bei Eingabe | 
| **Heute** | Nur für den aktuellen Tag geplante Reservierungen | 
| **Vergangenheiten** | Reservierungen, deren Endzeit bereits überschritten ist | 
| **Alle** | Alle Ihre Reservierungen ohne Filter | 
| **Nach Datum** | Öffnen Sie einen Kalender, um einen bestimmten Tag auszuwählen |

:::Tipp
Der Filter **„Vergangenheit“** berücksichtigt die genaue Endzeit, nicht nur den Tag. Eine Reservierung von heute, die bereits beendet ist, wird in „Vergangenheit“ korrekt angezeigt, auch wenn sie noch am selben Tag ist. 
:::

---

## Eine Reservierung stornieren

Um eine aktive Reservierung zu stornieren, drücken Sie die Stornierungsschaltfläche auf Ihrer Karte. Wenn Sie dies tun, geschieht automatisch Folgendes:

1. Die Reservierung wird aus dem System gelöscht. 
2. Die für diese Reservierung geplante Benachrichtigungserinnerung wird storniert. 
3. Wenn noch weitere Reservierungen anstehen, wird die Erinnerung auf die nächste verschoben.

Sie müssen nichts manuell tun, um Benachrichtigungen zu verwalten: Die App kümmert sich um alles, wenn Sie kündigen.

:::Vorsicht
Die Kündigung ist dauerhaft. Nach der Stornierung müssen Sie eine neue Reservierung vornehmen, wenn Sie den Automaten zu diesem Zeitpunkt erneut nutzen möchten, sofern dieser noch verfügbar ist. 
:::

---

## Exportieren Sie eine Reservierung in den Telefonkalender

Jede Reservierungskarte enthält in der oberen rechten Ecke ein Kalendersymbol. Beim Drücken:

1. Es erscheint ein Bestätigungsdialog mit den Reservierungsdaten (Maschine, Datum und Uhrzeit). 
2. Wenn Sie bestätigen, öffnet die App die Kalenderanwendung auf Ihrem Telefon mit allen bereits ausgefüllten Daten: Titel mit dem Namen der Maschine, Datum, Startzeit und Endzeit. 
3. Bestätigen Sie in Ihrer Kalender-App, um das Ereignis zu speichern.

Diese Funktion ist sowohl für zukünftige Reservierungen (visuelles Erinnern an Sie im Kalender) als auch für vergangene Reservierungen (Verfolgen einer historischen Aufzeichnung Ihrer Aktivitäten) nützlich.

:::Info
Beim Exportieren wird die Standardkalender-App Ihres Telefons geöffnet. Wenn Sie mehrere Kalender-Apps installiert haben, werden Sie vom System gefragt, welche Sie verwenden möchten. 
:::

---

## Administratoransicht

Wenn Ihr Konto über Administratorrechte verfügt, sehen Sie auf dem Reservierungsbildschirm auch die Reservierungen **der übrigen Benutzer**, nicht nur Ihre. Auf jeder Karte ist außerdem der Name des Benutzers vermerkt, der die Reservierung vorgenommen hat. Der Administrator kann jede aktive Reservierung stornieren, nicht nur seine eigene.
