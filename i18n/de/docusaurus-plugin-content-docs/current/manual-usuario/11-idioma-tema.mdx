# Sprache und visuelles Thema

Mit Trainium können Sie zwei Aspekte des Erscheinungsbilds der App anpassen: die **Sprache** der Benutzeroberfläche und das **visuelle Thema** (dunkel oder hell). Beide Einstellungen werden automatisch gespeichert und bei jedem Öffnen der App gespeichert.

---

## Ändern Sie die Sprache

### Automatische Erkennung

Wenn Sie Trainium zum ersten Mal installieren, erkennt die App die Sprache Ihres Telefons und verwendet sie automatisch. Wenn Ihr Telefon auf Spanisch ist, wird die App auf Spanisch angezeigt. Wenn es auf Englisch ist, auf Englisch usw. mit den übrigen unterstützten Sprachen.

### Ändern Sie die Sprache manuell

Wenn Sie eine andere Sprache als Ihr Telefon verwenden möchten:

1. Drücken Sie auf das **Globussymbol**, das in der oberen Ecke des Hauptbildschirms (und anderer Bildschirme wie Maschinen, Reservierungen, Gerichte oder Profilbearbeitung) erscheint. 
2. Die Sprachauswahl wird mit der Liste der verfügbaren Optionen geöffnet. 
3. Klicken Sie auf die Sprache, die Sie verwenden möchten. 
4. Die Benutzeroberfläche ändert sich sofort, ohne dass die App neu gestartet werden muss.

:::Tipp
Der Sprachwechsel ist vollständig reversibel. Sie können es über das Globussymbol beliebig oft ändern. 
:::

## Ändern Sie das visuelle Thema

### So wechseln Sie zwischen dunklem und hellem Thema

Tippen Sie auf das **Sonnen- oder Mondsymbol**, das in der oberen Ecke mehrerer Bildschirme angezeigt wird:

- Wenn sich die App im **dunklen Design** befindet, sehen Sie das Sonnensymbol. Durch Drücken wird zum Lichtthema gewechselt. 
- Wenn die App im **Light-Design** ist, sehen Sie das Mondsymbol. Durch Drücken wird zum dunklen Thema gewechselt.

Die Änderung wird sofort auf die gesamte Schnittstelle angewendet.

### Beschreibung jedes Themas

**Dunkles Thema:**
Hintergründe in Mitternachtsblautönen (von `#0B1426` bis `#162347`). Ideal für den Einsatz in Umgebungen mit wenig Licht, beispielsweise nachts oder in dunklen Innenräumen. Reduziert die Belastung der Augen bei schlechten Lichtverhältnissen.

**Thema löschen:**
Weiche weiße Hintergründe (`#F0F4FF`). Bequemer im Freien oder in gut beleuchteten Umgebungen zu verwenden.

### Standardthema

Wenn Sie kein Design manuell ausgewählt haben, verwendet die App das **Systemdesign** Ihres Telefons. Wenn Ihr Telefon auf den Dunkelmodus eingestellt ist, verwendet Trainium das dunkle Design. Wenn Sie sich im Lichtmodus befinden, wird das Lichtthema verwendet.

Sobald Sie manuell ein Thema auswählen, wird diese Einstellung gespeichert und folgt nicht mehr Ihren Systemeinstellungen, bis Sie sie erneut ändern.
