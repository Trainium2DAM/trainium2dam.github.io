# Maschinen und Reserven

Im Bereich „Geräte“ können Sie die im Fitnessstudio verfügbare Ausrüstung überprüfen und das Gerät reservieren, das Sie in einem bestimmten Zeitfenster verwenden möchten.

---

## Siehe Maschinenkatalog

Drücken Sie auf dem Hauptbildschirm **„Maschinen“**, um die vollständige Liste der Geräte anzuzeigen. Während die Daten geladen werden, sehen Sie animierte graue Karten (Skeleton Loading), die durch die echten Karten ersetzt werden, sobald die Informationen eintreffen.

Jede Maschinenkarte zeigt:

- **Name** der Maschine. 
- **Kategorie** der Übungen (Cardio, Rücken, Beine, Brust usw.). 
- **Beschreibung** der Ausrüstung. 
- **Aktueller Status**, angezeigt durch ein farbiges Abzeichen:

| Abzeichen | Farbe | Bedeutung | 
|---|---|---| 
| **Verfügbar** | Grün | Die Maschine ist betriebsbereit und Sie können sie reservieren | 
| **Wartung** | Orange | Vorübergehend außer Betrieb, Reservierungen nicht möglich |

Wenn eine Maschine gewartet wird, zeigt die Karte auch das **Start- und Enddatum** des Zeitraums an, sodass Sie wissen, wann sie wieder verfügbar sein wird.

---

## Machen Sie eine Reservierung

Um eine Maschine zu reservieren, muss sie sich im Status **Verfügbar** befinden. Klicken Sie auf Ihre Karte, um den Vorgang zu starten:

### 1. Wählen Sie das Datum aus

Es öffnet sich ein Kalender, in dem Sie den Tag auswählen können, an dem Sie die Maschine nutzen möchten.

### 2. Wählen Sie die Startzeit

Nach Auswahl des Datums erscheint eine Zeitauswahl. Wählen Sie die Uhrzeit, zu der Sie beginnen möchten.

### 3. Endzeit festlegen (optional)

Die App berechnet die Endzeit automatisch, indem sie **30 Minuten** zur Startzeit addiert. Bei Bedarf können Sie diese Zeit anpassen. Beachten Sie jedoch, dass **pro Reservierung maximal 1 Stunde zulässig ist**. Wenn Sie versuchen, mehr Zeit zu reservieren, werden Sie von der App mit einer Fehlermeldung benachrichtigt.

### 4. Automatische Bestätigung der Verfügbarkeit

Vor dem Speichern der Reservierung prüft die App zwei Bedingungen:

- Dass die Maschine in diesem Zeitfenster nicht bereits von einem anderen Benutzer reserviert ist. 
- Dass Sie im selben Streifen keine weitere Reservierung haben.

Wenn ein Konflikt vorliegt, wird eine Meldung mit der Begründung angezeigt und Sie können ein anderes Datum oder eine andere Uhrzeit auswählen.

### 5. Bestätigte Reservierung

Wenn keine Konflikte vorliegen, wird die Reservierung gespeichert und **automatisch eine Erinnerung geplant**, um Sie vor der Startzeit zu benachrichtigen. Die Vorankündigung der Erinnerung ist diejenige, die Sie in „Mein Profil“ konfiguriert haben (standardmäßig 30 Minuten).

Am unteren Bildschirmrand erscheint ein Hinweis mit der Option **„Abbrechen“**. Wenn Sie einen Fehler gemacht haben, können Sie diese Taste einige Sekunden lang drücken, um die Reservierung sofort rückgängig zu machen, ohne zu „Meine Reservierungen“ gehen zu müssen.

:::Info
Es ist nicht möglich, zwei Reservierungen im selben Zeitfenster vorzunehmen, selbst wenn diese auf unterschiedlichen Maschinen erfolgen. Planen Sie Ihre Sitzungen gut, um Konflikte zu vermeiden. 
:::

## Maschinen in Wartung

Wenn der Administrator eine Maschine für einen bestimmten Zeitraum in die Wartung versetzt, passiert Folgendes:

- Die Maschine erscheint mit der orangefarbenen Plakette **„Wartung“** und die Reservierungsschaltfläche ist deaktiviert. 
- Wenn Sie in diesem Zeitraum eine Reservierung hatten, wird diese automatisch storniert. 
- Wenn Sie die App öffnen, sehen Sie einen Hinweis mit den betroffenen Maschinen und Reservierungen.

Nach Ablauf des Wartungszeitraums erscheint die Maschine wieder als **Verfügbar** und Sie können sie erneut reservieren.

---

## Reservierungsbeschränkungen

| Einschränkung | Detail | 
|---|---| 
| Maximale Dauer | 1 Stunde pro Reservierung | 
| Überlappende Zeitpläne | Sie können nicht zwei Reservierungen gleichzeitig haben | 
| Maschinen in der Wartung | Sie können während des Wartungszeitraums nicht reserviert werden | 
| Internetverbindung | Erforderlich, um die Verfügbarkeit zu prüfen und Reservierungen zu bestätigen |
