# Erinnerungsbenachrichtigungen

Trainium kann Ihnen eine Benachrichtigung auf Ihr Mobiltelefon senden, um Sie daran zu erinnern, dass Sie eine bevorstehende Reservierung haben. So vergisst du keine Trainingseinheit im Fitnessstudio.

---

## So funktionieren Erinnerungen

Wenn Sie eine Reservierung vornehmen, stellt die App automatisch einen Alarm auf Ihrem Telefon ein. Wenn die konfigurierte Zeit erreicht ist, erhalten Sie eine Benachrichtigung mit:

- **Titel:** „Trainium – Reservierungserinnerung“
- **Nachricht:** Name und Uhrzeit des Geräts (z. B. „Heimtrainer: 10:00 – 10:30“)

Sie müssen nichts tun, um Erinnerungen zu aktivieren: Sie werden geplant, wenn Sie jede Reservierung bestätigen.

---

## Benachrichtigungen konfigurieren

Sie können Benachrichtigungen über **Mein Profil → Profil bearbeiten** im Abschnitt **„Benachrichtigungen“** verwalten. Dort finden Sie zwei Steuerelemente:

### Benachrichtigungen ein- oder ausschalten

Mit einem **Schalter** können Sie Reservierungserinnerungen global ein- oder ausschalten. Wenn Sie es deaktivieren, erhalten Sie auch bei ausstehenden Reservierungen keine Benachrichtigung. Die Einstellung wird automatisch gespeichert.

### Konfigurieren Sie die Vorankündigung

Wenn Benachrichtigungen aktiviert sind, können Sie auswählen, wie lange im Voraus Sie die Erinnerung erhalten möchten, indem Sie eine der verfügbaren Schaltflächen drücken:

| Option | Wann kommt die Mitteilung? 
|---|---| 
| **10 Minuten** | 10 Minuten vor Beginn | 
| **15 Minuten** | 15 Minuten vorher | 
| **30 Minuten** *(Standard)* | 30 Minuten vorher | 
| **60 Minuten** | 1 Stunde vorher | 
| **120 Minuten** | 2 Stunden vorher |

Die Änderung wird automatisch gespeichert, sobald Sie eine Option drücken. Von da an verwendet die nächste Erinnerung die neuen Einstellungen.

---

## Benachrichtigungsberechtigung

Um Erinnerungen zu erhalten, benötigt die App die Erlaubnis, Benachrichtigungen auf Ihrem Telefon anzuzeigen.

**Auf Android 13 oder höher:** Wenn Sie Trainium zum ersten Mal öffnen, werden Sie vom System gefragt, ob Sie Benachrichtigungen zulassen möchten. Drücken Sie **„Zulassen“**, um sie zu aktivieren.

**Wenn Sie die Berechtigung abgelehnt haben:** Sie können sie später in den Telefoneinstellungen aktivieren:

1. Öffnen Sie die **Einstellungen** Ihres Telefons. 
2. Gehen Sie zu **Anwendungen**. 
3. Suchen Sie und klicken Sie auf **Trainium**. 
4. Gehen Sie zu **Benachrichtigungen**. 
5. Aktivieren Sie den Schalter.

:::Info
Wenn Sie keine Benachrichtigungsberechtigung erteilen, funktioniert die App normal. Das bedeutet lediglich, dass Sie vor Ihrer Reservierung keine Erinnerungsbenachrichtigungen erhalten. 
:::

---

## Automatische Neuplanung

Die App hält die Erinnerung an Ihre nächste Reservierung immer auf dem neuesten Stand. Das bedeutet:

- Wenn Sie eine **neue Reservierung** vornehmen, wird Ihre nächste zukünftige Reservierung ausgewertet und die Erinnerung für diese geplant. 
- Wenn Sie **eine Reservierung stornieren**, wird die Erinnerung für diese Reservierung gelöscht und automatisch für die nächste von Ihnen neu geplant.

Sie müssen Erinnerungen zu keinem Zeitpunkt manuell verwalten.

---

## Fragen zu Benachrichtigungen

**Kann ich eine Erinnerung erhalten, wenn mein Telefon stumm geschaltet ist?**
Ja. Die Erinnerung kommt als Standard-Systembenachrichtigung. Wenn Ihr Telefon stumm geschaltet ist oder sich im „Bitte nicht stören“-Modus befindet, erhalten Sie die stille Benachrichtigung, die Sie in der Statusleiste sehen.

**Was passiert, wenn das Telefon ausgeschaltet ist, wenn die Benachrichtigung eintrifft?**
Wenn das Telefon zum Zeitpunkt der Benachrichtigung ausgeschaltet war und Sie es später wieder einschalten, kommt die Benachrichtigung nicht an. Für Erinnerungen muss das Gerät zum Zeitpunkt der Auslösung eingeschaltet sein.

**Kann ich die Erinnerung erhalten, wenn die App geschlossen ist?**
Ja. Das Android-Alarmsystem funktioniert auch, wenn die App im Hintergrund geschlossen ist.
