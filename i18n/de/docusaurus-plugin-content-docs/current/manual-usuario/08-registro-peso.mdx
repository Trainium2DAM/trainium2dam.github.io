# Gewichtsaufzeichnung

Im Abschnitt **Gewichtsverfolgung** können Sie Ihr Körpergewicht täglich verfolgen und mithilfe eines Liniendiagramms visualisieren, wie es sich im Laufe der Zeit entwickelt.

---

## Greifen Sie auf die Registrierung zu

Drücken Sie auf dem Hauptbildschirm **„Gewichtsaufzeichnung“**. Sie sehen zwei Hauptbereiche:

- **Eingabefeld** – um das Gewicht des Tages hinzuzufügen (nur sichtbar, wenn Sie das Gewicht heute noch nicht registriert haben). 
- **Liste der Datensätze** – alle Ihre vorherigen Datensätze, sortiert nach Datum. 
- **Evolutionsdiagramm** – Visualisierung Ihrer Geschichte im Zeitleistenformat.

---

## Heutiges Gewicht hinzufügen

1. Geben Sie Ihr aktuelles Gewicht in das Textfeld ein. Akzeptiert Dezimalwerte (z. B. `73.5` oder `73,5`). 
2. Drücken Sie die Bestätigungstaste.

Der Datensatz wird automatisch mit dem heutigen Datum gespeichert. Sie müssen das Datum nicht manuell eingeben.

:::Info
Sie können nur **einen Peso pro Tag** registrieren. Wenn Sie heute bereits Ihr Gewicht eingegeben haben, verschwindet das Eingabefeld und Sie sehen den erfassten Wert in der Liste. Kommen Sie morgen wieder, um den nächsten Datensatz hinzuzufügen. 
:::

---

## Bearbeiten Sie einen vorhandenen Datensatz

Wenn Ihnen bei der Eingabe eines Werts ein Fehler unterlaufen ist, können Sie diesen korrigieren:

1. Suchen Sie in der Datensatzliste den Eintrag, den Sie korrigieren möchten. 
2. Drücken Sie auf das Bearbeitungssymbol (Bleistift) für diese Zeile. 
3. Das Feld wird im Bearbeitungsmodus aktiviert: Geben Sie den richtigen Wert ein. 
4. Bestätigen Sie, um die Änderung zu speichern.

---

## Einen Datensatz löschen

Wenn Sie einen Verlaufseintrag löschen möchten:

1. Suchen Sie den Eintrag in der Liste. 
2. Klicken Sie auf das Löschsymbol (Papierkorb) für diese Zeile. 
3. Der Datensatz wird dauerhaft gelöscht.

:::Vorsicht
Das Löschen eines Datensatzes ist dauerhaft. Wenn Sie einen Eintrag löschen und das Diagramm wichtige Referenzpunkte verliert, können Sie diese nicht wiederherstellen. 
:::

---

## Evolutionsdiagramm

Die Grafik zeigt Ihren Gewichtsverlauf im Zeitleistenformat:

- Die **horizontale Achse** stellt die Zeit dar (die Datensätze erscheinen chronologisch von links nach rechts geordnet). 
- Die **vertikale Achse** stellt das Gewicht in Kilogramm dar.
- Die **Mindest- und Höchstwerte** Ihres Verlaufs werden als Referenzbezeichnungen an den Enden der Achse angezeigt. 
- Jeder Punkt in der Grafik entspricht einer Aufzeichnung Ihres Verlaufs.

:::Info
Sie benötigen mindestens **zwei Datensätze**, damit das Diagramm angezeigt wird. Anhand eines einzelnen Punktes ist es nicht möglich, eine Entwicklungslinie zu zeichnen. 
:::

---

## Tipps für eine effektive Nachbereitung

- **Wiegen Sie sich immer zur gleichen Tageszeit** (idealerweise morgens auf nüchternen Magen), damit Vergleiche zwischen Tagen genauer sind. 
- **Notieren Sie es jeden Tag**, damit die Grafik eine kontinuierliche Entwicklung ohne Lücken zeigt. 
- **Lassen Sie sich nicht durch tägliche Schwankungen beunruhigen**: Das Gewicht kann aufgrund von Faktoren wie Flüssigkeitszufuhr, Salzkonsum oder dem Zeitpunkt des Menstruationszyklus von Tag zu Tag um 0,5–2 kg schwanken. Entscheidend ist der langfristige Trend.
