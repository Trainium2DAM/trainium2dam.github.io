# Xestión de reservas

A sección **"As miñas reservas"** permíteche consultar todas as túas reservas, filtralas por data ou estado, cancelar as que xa non necesitas e exportalas ao calendario do teu teléfono.

---

## Consulta as túas reservas

Preme **"As miñas reservas"** na pantalla principal. Todas as túas reservas cargaranse ordenadas por data (o máis próximo primeiro). Mentres se está cargando a lista, verá o efecto da carga do esqueleto.

Cada tarxeta de reserva mostra:

- Nome da **máquina** reservada. 
- **Data** da reserva. 
- **Hora de inicio e fin**. 
- Estado (activo ou pasado).

---

## Filtrar as reservas

Na parte superior da pantalla atoparás os filtros dispoñibles. Podes desprazalos horizontalmente se non caben todos na pantalla:

| Filtro | Que mostra | 
|---|---| 
| **Próximamente** | Reservas cuxa hora de finalización aínda non chegou. Vista predeterminada ao entrar | 
| **Hoxe** | Só reservas programadas para o día actual | 
| **Pasados** | Reservas cuxa hora de finalización xa pasou | 
| **Todos** | Todas as túas reservas sen ningún filtro | 
| **Por data** | Abre un calendario para escoller un día específico |

:::tip
O filtro **"Pasado"** ten en conta a hora exacta de finalización, non só o día. Unha reserva de hoxe que xa rematou aparecerá correctamente en "Pasados" aínda que aínda sexa o mesmo día. 
:::

---

## Cancelar unha reserva

Para cancelar unha reserva activa, preme o botón de cancelación que aparece na túa tarxeta. Cando fas isto, ocorre automaticamente o seguinte:

1. A reserva elimínase do sistema. 
2. Cancelarase o recordatorio de notificación programado para esa reserva. 
3. Se tes máis reservas próximas, o recordatorio reprogramarase para apuntar á seguinte.

Non necesitas facer nada manualmente para xestionar as notificacións: a aplicación encárgase de todo cando cancelas.

:::precaución
A cancelación é permanente. Unha vez cancelada, terás que facer unha nova reserva se queres volver utilizar esa máquina nese momento, sempre que aínda estea dispoñible. 
:::

---

## Exporta unha reserva ao calendario do teléfono

Cada tarxeta de reserva inclúe unha icona de calendario na esquina superior dereita. Cando se preme:

1. Aparece un diálogo de confirmación cos datos da reserva (máquina, data e hora). 
2. Se o confirmas, a aplicación abre a aplicación de calendario no teu teléfono con todos os datos xa cubertos: título co nome da máquina, data, hora de inicio e hora de finalización. 
3. Confirma na túa aplicación de calendario para gardar o evento.

Esta función é útil tanto para reservas futuras (recordándote visualmente no calendario) como para reservas pasadas (mantendo un rexistro histórico da túa actividade).

:::información
Ao exportar abre a aplicación de calendario predeterminada do teu teléfono. Se tes varias aplicacións de calendario instaladas, o sistema preguntarache cal queres usar. 
:::

---

## Vista de administrador

Se a túa conta ten permisos de administrador, na pantalla de reservas tamén verás as reservas de **o resto dos usuarios**, non só as túas. Cada tarxeta tamén inclúe o nome do usuario que fixo a reserva. O administrador pode cancelar calquera reserva activa, non só a súa.
