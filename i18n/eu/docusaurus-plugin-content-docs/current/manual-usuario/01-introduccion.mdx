# Trainium-en aurkezpena

Trainium Android aplikazio bat da zure gimnasioko jarduerak zure mugikorretik kudeatzeko. Harekin makinak erreserba ditzakezu, zure pisuaren bilakaera erregistratu, errezetak eta elikadura planak kontsultatu eta Premium harpidetza bat kontratatu funtzio guztietarako sarbidearekin.

## Zer egin dezakezu Trainium-ekin?

| atala | Zertarako balio du | 
|---|---| 
| **Makinak** | Kontsultatu ekipoen katalogoa eta erreserbatu ordu-tarteak | 
| **Nire erreserbak** | Ikusi, iragazi eta bertan behera utzi zure erreserbak; esportatu telefonoko egutegira | 
| **Platerak** | Arakatu errezetak, ikusi kaloriak eta prestatzeko denbora eta iradoki zurea | 
| **Pisu-erregistroa** | Idatzi zure eguneroko pisua eta ikusi zure bilakaera grafiko batean | 
| **Nire profila** | Editatu zure datu pertsonalak, argazkia eta konfiguratu jakinarazpenak | 
| **Premium** | Hilero, seihileko edo urteko harpidetza kontratatu |

## Aplikazioa erabiltzeko baldintzak

- 7.0 (Nougat) edo berriagoa duen Android telefonoa. 
- Interneteko konexio aktiboa. Datu-eragiketa guztiek (erreserbak, profila, platerak, pisua) konektibitatea behar dute, informazioa hodeian gordetzen baita.

## Eskuliburu honen egitura

Eskuliburu hau atal independenteetan banatuta dago. Hasieratik amaierara irakur dezakezu edo zuzenean behar duzun atalera joan:

1. **Lehen urratsak** — Sortu kontua, hasi saioa eta berreskuratu pasahitza
2. **Pantaila nagusia** — Aplikazioa nola antolatzen den behin barruan
3. **Makinak eta erreserbak** — Gimnasioko ekipamendua erreserbatu
4. **Erreserben kudeaketa** — Iragazi, bertan behera utzi eta esportatu erreserbak
5. **Jakinarazpenak** — Ezarri abisuak erreserba bakoitzaren aurretik
6. **Elikadura eta platerak** — Errezetak kontsultatu eta proposatu
7. **Pisuaren erregistroa** — Eguneko jarraipena eta bilakaera grafikoa
8. **Nire profila** — Editatu datuak, argazkiak eta kontuaren ezarpenak
9. **Premium Harpidetza** — Planak, kontratazioa eta jarraipena
10. **Hizkuntza eta gaia** — Aldatu hizkuntza eta itxura bisuala
11. **Ohiko galderak** — Galdera ohikoenen erantzunak
