# Язык и визуальная тема

Trainium позволяет вам настроить два аспекта внешнего вида приложения: **язык** интерфейса и **визуальную тему** (темную или светлую). Обе настройки сохраняются автоматически и запоминаются каждый раз, когда вы открываете приложение.

---

## Изменить язык

### Автоопределение

Когда вы устанавливаете Trainium в первый раз, приложение определяет язык вашего телефона и использует его автоматически. Если ваш телефон на испанском языке, приложение будет отображаться на испанском языке. Если на английском, то на английском и так далее с остальными поддерживаемыми языками.

### Изменить язык вручную

Если вы хотите использовать язык, отличный от языка вашего телефона:

1. Нажмите **значок глобуса**, который появляется в верхнем углу главного экрана (и других экранов, таких как машины, бронирование, блюда или редактирование профиля). 
2. Откроется окно выбора языка со списком доступных опций. 
3. Нажмите на язык, который вы хотите использовать. 
4. Интерфейс меняется мгновенно, без необходимости перезапускать приложение.

### Доступные языки

| Язык | Родное имя | 
|---|---| 
| испанский | испанский | 
| английский | английский | 
| Валенсиано | Валенсия | 
| каталанский | английский | 
| французский | французский | 
| Марокканский арабский | Степень | 
| баскский | баскский | 
| Гальего | Галего | 
| немецкий | немецкий | 
| Португальский | Португальский | 
| Русский | Русский | 
| китайский | 中文 |

Если языка вашего телефона нет в этом списке, приложение будет использовать **испанский** в качестве языка по умолчанию.

:::совет
Изменение языка полностью обратимо. Вы можете изменить его столько раз, сколько захотите, с помощью значка глобуса. 
:::

## Изменить визуальную тему

### Как переключиться между темной и светлой темой

Коснитесь **значка солнца или луны**, который появляется в верхнем углу нескольких экранов:

- Если приложение использует **тёмную тему**, вы увидите значок солнца. Нажатие переключает на светлую тему. 
- Если приложение имеет **светлую тему**, вы увидите значок луны. Нажатие переключает на темную тему.

Изменение применяется немедленно по всему интерфейсу.

### Описание каждой темы

**Тёмная тема:**
Фон в темно-синих тонах (от `#0B1426` до `#162347`). Идеально подходит для использования в условиях низкой освещенности, например, ночью или в темных помещениях. Снижает нагрузку на глаза в условиях низкой освещенности.

**Очистить тему:**
Мягкий белый фон (`#F0F4FF`). Более удобно использовать на открытом воздухе или в хорошо освещенных помещениях.

### Тема по умолчанию

Если вы не выбрали тему вручную, приложение использует **системную тему** вашего телефона. Если на вашем телефоне установлен темный режим, Trainium будет использовать темную тему. Если вы находитесь в светлом режиме, будет использоваться светлая тема.

Как только вы выберете тему вручную, эта настройка сохраняется и больше не соответствует настройкам вашей системы, пока вы не измените ее снова.
