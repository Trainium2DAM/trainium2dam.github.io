# Nutrició i plats

La secció de **Platos** és el catàleg de receptes i suggeriments nutricionals del gimnàs. Qualsevol usuari pot explorar els plats disponibles i proposar les pròpies receptes perquè siguin afegides al catàleg.

---

## Explorar el catàleg

Clica **"Plats"** a la pantalla principal per accedir al catàleg. Els plats es mostren un per un en format targeta gran, com un carrusel lineal. Fes servir les **fletxes de navegació** per passar al plat següent o tornar a l'anterior.

Només apareixen al catàleg els plats que han estat aprovats per un administrador. Els suggeriments pendents de revisió no són visibles per als usuaris estàndard.

---

## Informació de cada plat

Cada targeta de plat mostra:

| Dada | Descripció | 
|---|---| 
| **Foto** | Imatge del plat (si es va afegir en pujar-lo) | 
| **Nom** | Nom del plat o recepta | 
| **Calories** | Contingut calòric aproximat per ració | 
| **Temps de preparació** | Temps estimat (per exemple, 20 minuts) | 
| **Descripció / Recepta** | Ingredients, passos de preparació o altra informació rellevant |

---

## Suggerir un plat nou

Si vols proposar una recepta perquè sigui afegida al catàleg, prem el botó de suggeriment (icona de més o similar) i omple el formulari:

### Camps del formulari

| Camp | Obligatori | Descripció | 
|---|---|---| 
| **Nom del plat** | Sí | El títol amb què apareixerà al catàleg | 
| **Descripció / Recepta** | No | Ingredients, passos, consells de preparació... | 
| **Temps de preparació** | No | Escriviu el temps en text lliure, per exemple "30 minuts" | 
| **Calories** | No | Calories aproximades per ració | 
| **Foto** | No | Seleccioneu una imatge de la vostra galeria |

Quan hagis completat el formulari prem **"Envia suggeriment"**.

### Què passa després?

1. El teu suggeriment queda registrat amb estat **pendent de revisió**. 
2. L'administrador del gimnàs la revisa. 
3. Si l'aprova, el plat apareix al catàleg visible per a tots els usuaris. 
4. Si la rebutja, el plat s'elimina sense que rebis notificació (consulta amb l'administrador si en vols saber el motiu).

:::info 
No cal ser usuari Premium per suggerir plats. Aquesta funció està disponible per a tots els usuaris registrats. 
:::

---

## Bones pràctiques en suggerir plats

Perquè el teu suggeriment tingui més possibilitats de ser aprovat:

- Posa un **nom clar i descriptiu** que identifiqui bé el plat. 
- Inclou una recepta detallada amb ingredients i passos. Com més completa, més útil és per a altres usuaris. 
- Afegeix una **foto de qualitat** del plat acabat. Els suggeriments amb foto tenen més impacte visual al catàleg. 
- Indica les **calories** si les coneixes, encara que sigui de forma aproximada. 
- Assegureu-vos que el plat és apte per al context esportiu: receptes equilibrades, fonts de proteïna, carbohidrats de qualitat, etc.
