# Introducció a Trainium

Trainium és una aplicació Android per gestionar la teva activitat al gimnàs des del mòbil. Amb ella pots reservar màquines, registrar la teva evolució de pes, consultar receptes i plans de nutrició i contractar una subscripció Premium amb accés a totes les funcionalitats.

## Què pots fer amb Trainium?

| Secció | Per què serveix | 
|---|---| 
| **Màquines** | Consulteu el catàleg d'equipament i reserveu franges horàries | 
| **Les meves reserves** | Veure, filtrar i cancel·lar les teves reserves; exportar-les al calendari del telèfon | 
| **Plats** | Explorar receptes, veure calories i temps de preparació i suggerir les teves | 
| **Registre de pes** | Apuntar el pes diari i visualitzar la teva evolució en un gràfic | 
| **El meu Perfil** | Editeu les vostres dades personals, foto i configureu notificacions | 
| **Premium** | Contractar una subscripció mensual, semestral o anual |

## Requisits per utilitzar l'app

- Telèfon Android amb versió 7.0 (Nougat) o superior. 
- Connexió a Internet activa. Totes les operacions de dades (reserves, perfil, plats, pes) requereixen connectivitat, ja que la informació s'emmagatzema al núvol.

## Estructura d'aquest manual

Aquest manual està dividit en seccions independents. Pots llegir-lo de principi a fi o anar directament a la secció que necessitis:

1. **Primers passos** — Crear compte, iniciar sessió i recuperar contrasenya 
2. **Pantalla principal** — Com està organitzada l'app un cop dins 
3. **Màquines i reserves** — Reservar equipament del gimnàs 
4. **Gestió de reserves** — Filtrar, cancel·lar i exportar reserves 
5. **Notificacions** — Configura recordatoris abans de cada reserva 
6. **Nutrició i plats** — Consultar i suggerir receptes 
7. **Registre de pes** — Seguiment diari i gràfic d'evolució 
8. **El meu perfil** — Edita dades, fotografia i paràmetres de compte 
9. **Subscripció Premium** — Plans, contractació i seguiment 
10. **Idioma i tema** — Canviar idioma i aparença visual 
11. **Preguntes freqüents** — Respostes als dubtes més habituals
