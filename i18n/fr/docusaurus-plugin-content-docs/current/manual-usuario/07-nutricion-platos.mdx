# Nutrition et plats

La section **Plats** est le catalogue de recettes et de suggestions nutritionnelles du gymnase. Tout utilisateur peut explorer les plats disponibles et proposer ses propres recettes à ajouter au catalogue.

---

## Explorez le catalogue

Appuyez sur **"Plats"** sur l'écran principal pour accéder au catalogue. Les plats sont présentés un à un dans un grand format de carte, à la manière d'un carrousel linéaire. Utilisez les **flèches de navigation** pour passer au plat suivant ou revenir au précédent.

Seuls les plats **approuvés par un administrateur** apparaissent dans le catalogue. Les suggestions en attente de révision ne sont pas visibles pour les utilisateurs standard.

---

## Informations sur chaque plat

Chaque fiche de plat montre :

| Données | Descriptif | 
|---|---| 
| **Photo** | Image du plat (si ajoutée lors du téléchargement) | 
| **Nom** | Nom du plat ou de la recette | 
| **Calories** | Teneur approximative en calories par portion | 
| **Temps de préparation** | Durée estimée (par exemple, "20 minutes") | 
| **Description / Recette** | Ingrédients, étapes de préparation ou autres informations pertinentes |

---

## Proposer un nouveau plat

Si vous souhaitez proposer une recette à ajouter au catalogue, appuyez sur le bouton de suggestion (icône plus ou similaire) et remplissez le formulaire :

### Champs du formulaire

| Champ | Obligatoire | Descriptif | 
|---|---|---| 
| **Nom du plat** | Oui | Le titre avec lequel il apparaîtra dans le catalogue | 
| **Description / Recette** | Non | Ingrédients, étapes, conseils de préparation... | 
| **Temps de préparation** | Non | Écrivez l'heure en texte libre, par exemple « 30 minutes » | 
| **Calories** | Non | Calories approximatives par portion | 
| **Photo** | Non | Sélectionnez une image dans votre galerie |

Lorsque vous avez rempli le formulaire, appuyez sur **"Envoyer la suggestion"**.

### Que se passe-t-il ensuite ?

1. Votre suggestion est enregistrée avec le statut **en attente d'examen**. 
2. Le directeur du gymnase la contrôle. 
3. Si vous l'approuvez, le plat apparaît dans le catalogue visible par tous les utilisateurs. 
4. Si vous le refusez, le plat est supprimé sans que vous receviez de notification (vérifiez auprès de l'administrateur si vous souhaitez connaître la raison).

:::infos
Vous n'avez pas besoin d'être un utilisateur Premium pour suggérer des plats. Cette fonctionnalité est disponible pour tous les utilisateurs enregistrés. 
:::

## Bonnes pratiques pour suggérer des plats

Pour donner à votre suggestion une meilleure chance d’être approuvée :

- Mettez un **nom clair et descriptif** qui identifie clairement le plat. 
- Comprend une **recette détaillée** avec les ingrédients et les étapes. Plus il est complet, plus il est utile aux autres utilisateurs. 
- Ajoutez une **photo de qualité** du plat fini. Les suggestions avec photos ont plus d'impact visuel dans le catalogue. 
- Indiquez les **calories** si vous les connaissez, même si c'est approximatif. 
- Assurez-vous que le plat est **adapté au contexte sportif** : recettes équilibrées, sources de protéines, glucides de qualité, etc.
