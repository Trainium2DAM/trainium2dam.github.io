# Écran principal

Après vous être connecté, vous serez redirigé vers l’écran de votre profil, qui est le menu principal de Trainium. De là, vous accédez à toutes les sections avec une seule touche.

---

## Ce que vous verrez sur votre écran principal

En haut apparaît votre **nom** et, le cas échéant, l'un de ces badges :

- **Premium** (en or) — vous disposez d'un abonnement actif. À côté du badge, vous verrez les jours qu'il vous reste. 
- **Administrateur** — votre compte dispose d'autorisations de gestion de salle de sport.

Si vous êtes un utilisateur Premium, lorsque l'abonnement expirera le même jour, l'indicateur affichera **"Expire aujourd'hui"** pour vous rappeler de le renouveler.

---

## Boutons de navigation

Depuis l'écran principal, vous pouvez accéder à :

| Bouton | Descriptif | 
|---|---| 
| **Machines** | Consulter le catalogue du matériel et faire des réservations | 
| **Mes réservations** | Consultez, filtrez et gérez toutes vos réservations | 
| **Plats** | Explorez le catalogue de recettes et proposez la vôtre | 
| **Record de poids** | Notez votre poids quotidien et vérifiez vos progrès | 
| **Mon profil** | Modifiez vos données, changez la photo et ajustez les notifications | 
| **Déconnexion** | Déconnectez-vous en toute sécurité |

---

## Icônes d'en-tête

Dans le coin supérieur de l’écran principal, vous trouverez deux icônes :

- **Icône Soleil/Lune** — bascule entre le thème clair et le thème sombre. 
- **Icône Globe** : ouvre le sélecteur de langue.

Ces mêmes icônes sont également disponibles sur d'autres écrans principaux de l'application.

---

## Avis d'entretien

Si l'administrateur a mis en maintenance une machine qui affecte l'une de vos réservations, lorsque vous ouvrez l'application, vous verrez une notification contextuelle avec les noms des machines concernées et les réservations qui ont été automatiquement annulées. Cet avis apparaît **une seule fois par session** pour ne pas être gênant.
