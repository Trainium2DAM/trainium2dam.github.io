# Commencer

## Ouvrez l'application pour la première fois

Lorsque vous ouvrez Trainium, vous verrez une courte animation de bienvenue avec le logo de l'application. Après quelques secondes, vous arriverez à l'écran d'accueil, où vous pourrez choisir entre :

- **Accueil** — pour accéder à un compte existant. 
- **Créer un nouveau compte** — pour vous inscrire en tant que nouvel utilisateur.

---

## Créer un nouveau compte

Si c'est la première fois que vous utilisez Trainium, vous devez vous inscrire. Appuyez sur **"Créer un nouveau compte"** et remplissez le formulaire avec les informations suivantes :

| Champ | Descriptif | 
|---|---| 
| **Nom complet** | Votre nom tel qu'il apparaîtra dans l'application | 
| **DNI / NIE / Passeport** | Il s'agit de votre identifiant d'accès. Vous en aurez besoin à chaque fois que vous vous connecterez | 
| **Courriel** | Il sert uniquement à récupérer le mot de passe si vous l'oubliez | 
| **Téléphone** | Champ de contact (facultatif) | 
| **Mot de passe** | Choisissez un mot de passe fort et enregistrez-le en lieu sûr |

Lorsque vous avez rempli tous les champs, appuyez sur **"Créer un compte"**. L'application vérifiera que votre identifiant n'est pas déjà enregistré et, si tout est correct, vous entrerez directement dans l'application sans avoir à vous reconnecter.

:::astuce
Notez votre pièce d'identité dans un endroit accessible. C'est l'identifiant que l'application vous demandera à chaque fois que vous vous connecterez. 
:::

---

##Se connecter

Si vous possédez déjà un compte, appuyez sur **"Démarrer"** sur l'écran d'accueil et saisissez votre **ID** et votre **mot de passe**.

Une fois connecté, **la session reste active pendant 7 jours**. Si vous ouvrez l'application pendant cette période, vous accéderez directement à votre profil sans avoir à saisir à nouveau vos données.

---

## Récupérer le mot de passe

Si vous ne vous souvenez pas de votre mot de passe, appuyez sur **"Mot de passe oublié ?"** sur l'écran de connexion. Le processus de récupération se déroule en deux étapes :

### Étape 1 — Vérifiez votre identité

Saisissez votre **DNI** et votre **e-mail**. L'application vérifiera que les deux informations correspondent à celles de votre compte.

- S'ils correspondent, vous passerez automatiquement à la deuxième étape. 
- S'ils ne correspondent pas, vérifiez que vous utilisez bien l'identifiant et l'email avec lesquels vous vous êtes inscrit.

### Étape 2 — Définir le nouveau mot de passe

Saisissez votre **nouveau mot de passe** et confirmez-le en le saisissant une seconde fois. S'ils correspondent tous les deux, le mot de passe sera mis à jour et vous pourrez vous connecter immédiatement avec celui-ci.

:::attention
Si vous ne vous souvenez pas de l'e-mail avec lequel vous vous êtes inscrit, contactez l'administrateur de votre salle de sport pour vous aider à retrouver l'accès à votre compte. 
:::

---

## Se déconnecter

Pour vous déconnecter de votre compte, accédez à l'écran de profil et appuyez sur **"Déconnexion"**. La prochaine fois que vous ouvrirez l'application, vous devrez saisir à nouveau votre identifiant et votre mot de passe.

Si vous fermez simplement l'application sans vous déconnecter, la session restera active et vous vous connecterez directement pendant les 7 prochains jours.
