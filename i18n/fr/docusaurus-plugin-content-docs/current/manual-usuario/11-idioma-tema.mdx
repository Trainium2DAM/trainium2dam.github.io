# Langage et thème visuel

Trainium vous permet de personnaliser deux aspects de l'apparence de l'application : le **langage** de l'interface et le **thème visuel** (sombre ou clair). Les deux préférences sont enregistrées automatiquement et mémorisées à chaque fois que vous ouvrez l'application.

---

## Changer la langue

### Détection automatique

Lorsque vous installez Trainium pour la première fois, l'application détecte la langue de votre téléphone et l'utilise automatiquement. Si votre téléphone est en espagnol, l'application apparaîtra en espagnol. Si c'est en anglais, en anglais, et ainsi de suite avec les autres langues prises en charge.

### Changer la langue manuellement

Si vous souhaitez utiliser une langue différente de celle de votre téléphone :

1. Appuyez sur l'**icône globe** qui apparaît dans le coin supérieur de l'écran principal (et d'autres écrans tels que les machines, les réservations, les plats ou l'édition de profil). 
2. Le sélecteur de langue s'ouvre avec la liste des options disponibles. 
3. Cliquez sur la langue que vous souhaitez utiliser. 
4. L'interface change instantanément, sans qu'il soit nécessaire de redémarrer l'application.

### Langues disponibles

| Langue | Nom natif | 
|---|---| 
| Espagnol | Espagnol | 
| Anglais | Anglais | 
| Valence | Valence | 
| catalan | Anglais | 
| français | français | 
| Arabe marocain | Diplôme | 
| Basque | Basque | 
| Gallego | Galégo | 
| Allemand | Allemand | 
| Portugais | Portugais | 
| Russe | Russe | 
| Chinois | Chine |

Si la langue de votre téléphone ne figure pas dans cette liste, l'application utilisera l'**espagnol** comme langue par défaut.

:::astuce
Le changement de langue est complètement réversible. Vous pouvez le modifier autant de fois que vous le souhaitez depuis l'icône globe. 
:::

## Changer le thème visuel

### Comment basculer entre le thème sombre et clair

Appuyez sur l'**icône du soleil ou de la lune** qui apparaît dans le coin supérieur de plusieurs écrans :

- Si l'application est en **thème sombre**, vous verrez l'icône du soleil. En appuyant dessus, vous passez au thème lumineux. 
- Si l'application est en **thème clair**, vous verrez l'icône de la lune. En appuyant dessus, vous passez au thème sombre.

Le changement est appliqué immédiatement dans toute l’interface.

### Description de chaque sujet

**Thème sombre :**
Fonds dans les tons bleu nuit (de `#0B1426` à `#162347`). Idéal pour une utilisation dans des environnements faiblement éclairés, comme la nuit ou dans des intérieurs sombres. Réduit la fatigue oculaire dans des conditions de faible éclairage.

**Thème clair :**
Fonds blancs doux (`#F0F4FF`). Plus confortable à utiliser à l’extérieur ou dans des environnements bien éclairés.

### Thème par défaut

Si vous n'avez pas choisi de thème manuellement, l'application utilise le **thème système** de votre téléphone. Si votre téléphone est réglé en mode sombre, Trainium utilisera le thème sombre. Si vous êtes en mode clair, il utilisera le thème clair.

Une fois que vous avez choisi manuellement un thème, cette préférence est enregistrée et ne suit plus les paramètres de votre système jusqu'à ce que vous la modifiiez à nouveau.
