# Notifications de rappel

Trainium peut vous envoyer une notification sur votre mobile pour vous rappeler que vous avez une réservation à venir. De cette façon, vous n'oublierez aucune séance à la salle de sport.

---

## Comment fonctionnent les rappels

Lorsque vous effectuez une réservation, l'application déclenche automatiquement une alarme sur votre téléphone. Lorsque l'heure configurée arrive, vous recevez une notification avec :

- **Titre :** "Trainium — Rappel de réservation"
- **Message :** nom et heure de la machine (par exemple "Vélo d'appartement : 10h00 – 10h30")

Vous n'avez rien à faire pour activer les rappels : ils sont programmés lorsque vous confirmez chaque réservation.

---

## Configurer les notifications

Vous pouvez gérer les notifications depuis **Mon profil → Modifier le profil**, dans la section **"Notifications"**. Vous y trouverez deux contrôles :

### Activer ou désactiver les notifications

Un **switch** vous permet d'activer ou de désactiver les rappels de réservation à l'échelle mondiale. Si vous le désactivez, vous ne recevrez aucune notification même si vous avez des réservations en attente. Le paramètre est enregistré automatiquement.

### Configurer le préavis

Lorsque les notifications sont activées, vous pouvez choisir combien de temps à l'avance vous souhaitez recevoir le rappel en appuyant sur l'un des boutons disponibles :

| Options | Quand arrive l’avis ? 
|---|---| 
| **10 minutes** | 10 minutes avant l'heure de début | 
| **15 minutes** | 15 minutes avant | 
| **30 minutes** *(par défaut)* | 30 minutes avant | 
| **60 minutes** | 1 heure avant | 
| **120 minutes** | 2 heures avant |

La modification est automatiquement enregistrée dès que vous appuyez sur une option. Dès lors, le prochain rappel utilisera les nouveaux paramètres.

---

## Autorisation de notification

Pour recevoir des rappels, l'application doit avoir l'autorisation d'afficher des notifications sur votre téléphone.

**Sur Android 13 ou supérieur :** La première fois que vous ouvrez Trainium, le système vous demande si vous souhaitez autoriser les notifications. Appuyez sur **"Autoriser"** pour les activer.

**Si vous avez refusé l'autorisation :** Vous pourrez l'activer ultérieurement à partir des paramètres du téléphone :

1. Ouvrez les **Paramètres** de votre téléphone. 
2. Accédez à **Applications**. 
3. Recherchez et cliquez sur **Trainium**. 
4. Accédez à **Notifications**. 
5. Activez l'interrupteur.

:::infos
Si vous n'accordez pas l'autorisation de notification, l'application fonctionne normalement. Cela signifie simplement que vous ne recevrez pas de rappels avant vos réservations. 
:::

---

## Reprogrammation automatique

L'application garde toujours à jour le rappel de votre prochaine réservation. Cela signifie que :

- Lorsque vous effectuez une **nouvelle réservation**, votre prochaine future réservation est évaluée et le rappel est programmé pour celle-là. 
- Lorsque vous **annulez une réservation**, le rappel de cette réservation est supprimé et automatiquement reprogrammé pour la prochaine que vous avez.

Vous n'avez à aucun moment besoin de gérer les rappels manuellement.

---

## Questions sur les notifications

**Puis-je recevoir un rappel si mon téléphone est en mode silencieux ?**
Oui. Le rappel arrive sous forme de notification système standard. Si votre téléphone est en mode silencieux ou en mode « Ne pas déranger », vous recevrez la notification silencieuse que vous voyez dans la barre d'état.

**Que se passe-t-il si le téléphone est éteint lorsque l'avis arrive ?**
Si le téléphone était éteint au moment de la notification et que vous le rallumez plus tard, la notification n'arrivera pas. Les rappels nécessitent que l'appareil soit allumé au moment du déclenchement.

**Puis-je recevoir le rappel si l'application est fermée ?**
Oui. Le système d'alarme Android fonctionne même si l'application est fermée en arrière-plan.
