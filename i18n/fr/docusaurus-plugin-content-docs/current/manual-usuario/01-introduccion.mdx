# Introduction à Trainium

Trainium est une application Android pour gérer votre activité en salle depuis votre mobile. Avec lui, vous pouvez réserver des machines, enregistrer votre évolution de poids, consulter des recettes et des plans nutritionnels et souscrire un abonnement Premium avec accès à toutes les fonctionnalités.

## Que pouvez-vous faire avec Trainium ?

| Rubrique | A quoi ça sert | 
|---|---| 
| **Machines** | Consulter le catalogue du matériel et réserver des plages horaires | 
| **Mes réservations** | Consultez, filtrez et annulez vos réservations ; exportez-les vers le calendrier du téléphone | 
| **Plats** | Parcourez les recettes, consultez les calories et le temps de préparation, et suggérez les vôtres | 
| **Record de poids** | Notez votre poids journalier et visualisez votre évolution sur un graphique | 
| **Mon profil** | Modifiez vos données personnelles, votre photo et configurez les notifications | 
| **Premium** | Contracter un abonnement mensuel, semestriel ou annuel |

## Conditions requises pour utiliser l'application

- Téléphone Android avec version 7.0 (Nougat) ou supérieure. 
-Connexion Internet active. Toutes les opérations sur les données (réservations, profil, plats, poids) nécessitent une connectivité, car les informations sont stockées dans le cloud.
