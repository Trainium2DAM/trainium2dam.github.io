# Mon profil

La rubrique **"Mon Profil"** centralise tout ce qui concerne votre compte : données personnelles, photo, paramètres de notification, statut de l'abonnement et historique des paiements.

---

## Accéder à la modification du profil

Appuyez sur **"Mon profil"** sur l'écran principal pour afficher votre profil. De là, appuyez sur **"Modifier le profil"** pour accéder au formulaire d'édition.

---

## Données personnelles

Dans le formulaire d'édition, vous pouvez modifier les champs suivants :

| Champ | Descriptif | 
|---|---| 
| **Nom complet** | Votre nom tel qu'il apparaît dans l'application | 
| **Courriel** | Adresse utilisée pour récupérer le mot de passe | 
| **Téléphone** | Numéro de contact | 
| **Mot de passe** | Laissez-le vide si vous ne souhaitez pas le modifier ; écrivez le nouveau si vous souhaitez le mettre à jour |

Lorsque vous avez effectué les modifications souhaitées, appuyez sur **"Enregistrer les modifications"**. Vous verrez une confirmation que les données ont été mises à jour avec succès.

:::infos
L'identifiant ne peut pas être modifié depuis l'application. Si vous devez modifier l'identifiant associé à votre compte, contactez l'administrateur du gymnase. 
:::

---

## Photo de profil

Votre photo apparaît sous la forme d'un cercle en haut de l'écran de profil. Pour le modifier ou en ajouter un :

1. Cliquez sur l'avatar circulaire sur l'écran d'édition. 
2. Choisissez la source de l'image :
- **Galerie** : sélectionnez une photo existante sur votre téléphone. 
- **Fichiers** : recherche l'image dans le stockage de l'appareil. 
3. L'image est automatiquement recadrée dans un format circulaire et enregistrée avec vos données.

Pour **supprimer la photo** et revenir à l'avatar générique, appuyez sur **"Supprimer la photo"**.

:::astuce
Utilisez une photo bien éclairée et montrant clairement votre visage. L'administrateur l'utilise pour vous identifier en tant que membre du gymnase. 
:::

---

## Paramètres de notification

Sur l'écran d'édition du profil, vous trouverez la section **"Notifications"**, où vous pourrez configurer combien de temps à l'avance vous souhaitez recevoir le rappel de vos réservations.

Les options disponibles sont : **10, 15, 30, 60 et 120 minutes** avant l'heure de début. La valeur par défaut est de 30 minutes. Appuyez sur l'option que vous préférez et elle sera enregistrée automatiquement.

Pour plus d'informations sur le fonctionnement des notifications, consultez la section [Notifications de rappel](./06-notifications).

---

## Statut du compte et abonnement

Sur l'écran de profil, vous verrez votre **statut de compte** :

- **Standard** — compte de base sans abonnement Premium. 
- **Premium** — abonnement actif. Le nombre de jours restants ainsi que les dates de début et de fin de l'abonnement sont affichés.

Si votre compte est standard, vous verrez le bouton **"Mettre à niveau"** pour vous inscrire à Premium. Si vous avez déjà Premium actif et que vous souhaitez le renouveler avant son expiration, vous pouvez également le faire à partir d'ici.

---

## Historique des paiements

Depuis l'écran d'édition du profil, vous pouvez accéder à l'**historique des paiements**, qui répertorie tous les abonnements Premium que vous avez souscrits. Chaque entrée affiche :

- **Type de plan** — Mensuel, semestriel ou annuel. 
- **Mode de paiement** — Carte ou Bizum. 
- **Montant** — en euros. 
- **Date** — jour où le paiement a été effectué.

Les paiements apparaissent du plus récent au plus ancien.
