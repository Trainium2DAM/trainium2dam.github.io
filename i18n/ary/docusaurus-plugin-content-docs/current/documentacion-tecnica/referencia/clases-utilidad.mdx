## إيماج يوتيلز

**فيشيرو:** `ImageUtils.kt`

يحتوي على وظيفتين للتحويل بين صيغ الصور. يتم استخدامه لتخزين صور الملفات الشخصية وصور الآلة وصور الأطباق في Supabase.

### uriToBase64

يحول URI للصورة المحددة من معرض الجهاز إلى سلسلة Base64 مضغوطة بتنسيق JPEG.

```kotlin
fun uriToBase64(uri: Uri, contentResolver: ContentResolver): String?
```

| المعلمة | اكتب | الوصف | 
|---|---|---| 
| `uri` | `Uri` | URI للصورة التي حددها المستخدم | 
| `contentResolver` | `ContentResolver` | حل من السياق لفتح دفق الصور |

**قيمة الإرجاع:** سلسلة Base64 للصورة المضغوطة، أو `null` في حالة حدوث أي استثناء.

**العملية الداخلية:**
1. افتح `InputStream` لمعرف URI باستخدام `contentResolver.openInputStream(uri)`. 
2. قم بفك تشفير الدفق إلى `Bitmap` باستخدام `BitmapFactory.decodeStream`. 
3. قم بضغط الصورة النقطية إلى JPEG بجودة `AppConfig.IMAGE_QUALITY` (70) في `ByteArrayOutputStream`. 
4. قم بتشفير مصفوفة البايت الناتجة إلى Base64 باستخدام `Base64.encodeToString(byteArray, Base64.DEFAULT)`.

### decodeBase64ToBitmap

تحويل سلسلة Base64 مرة أخرى إلى كائن `Bitmap`.

```kotlin
fun decodeBase64ToBitmap(base64Str: String): Bitmap?
```

| المعلمة | اكتب | الوصف | 
|---|---|---| 
| `base64Str` | `String` | سلسلة Base64 تمثل الصورة |

**قيمة الإرجاع:** `Bitmap` تم فك تشفيرها، أو `null` إذا كانت السلسلة غير صالحة Base64 أو حدث أي استثناء.

## مدير الموضوع

**فيشيرو:** `ThemeManager.kt`

يدير استمرارية تفضيلات السمة (فاتحة/غامقة) باستخدام تفضيلات Jetpack DataStore.

```kotlin
class ThemeManager(private val context: Context) {
    val isDarkMode: Flow<Boolean?>
    suspend fun setDarkMode(isDark: Boolean)
}
```

`isDarkMode` يخرج `null` عند التشغيل الأول. المُجمع في `MainActivity.kt` يعامل `null` على أنه "يستخدم سمة النظام".

## مدير إعدادات الإشعارات

**فيشيرو:** `NotificationSettingsManager.kt`

يستمر كل من إشعار الحجز المسبق الذي اختاره المستخدم وحالة التنشيط العامة للإشعارات في DataStore.

```kotlin
class NotificationSettingsManager(private val context: Context) {
    val minutesBefore: Flow<Int>
    val notificationsEnabled: Flow<Boolean>

    suspend fun setMinutesBefore(minutes: Int)
    suspend fun setNotificationsEnabled(enabled: Boolean)

    companion object {
        const val DEFAULT_MINUTES = 30
        val OPTIONS = listOf(10, 15, 30, 60, 120)
    }
}
```

`minutesBefore` يقوم بإخراج القيمة الحالية (الافتراضي: 30 دقيقة). `notificationsEnabled` يخرج `true` بشكل افتراضي. توجد واجهة مستخدم الإعداد في `EditProfileScreen`. راجع [Notificaciones de reservas](/docs/funcionalidades/notificaciones) للاستخدام الكامل.

## مدير اللغة

**فيشيرو:** `data/i18n/LanguageManager.kt`

يستمر في اللغة التي حددها المستخدم من خلال DataStore.

```kotlin
class LanguageManager(private val context: Context) {
    val currentLanguage: Flow<AppLanguage?>
    suspend fun setLanguage(language: AppLanguage)
    fun detectSystemLanguage(): AppLanguage
}
```

`currentLanguage` يخرج `null` حتى يقوم المستخدم بتحديد اللغة. يقوم `detectSystemLanguage()` بتعيين `Locale.getDefault()` إلى أقرب `AppLanguage`، مع `SPANISH` كبديل. راجع [Internacionalización](/docs/arquitectura/i18n) للتعرف على النظام الكامل.

## مدير قاعدة البيانات

**فيشيرو:** `DatabaseAdmin.kt`

Singleton الذي يحدد اتصال JDBC المباشر بقاعدة بيانات MySQL/MariaDB المحلية. تتوافق هذه الفئة مع الاختصار الذي تم استخدامه أثناء المراحل الأولى من التطوير.

**هذه الفئة غير نشطة في إصدار الإنتاج**; يتم تنفيذ جميع عمليات البيانات من خلال `SupabaseClient`. وهي محفوظة في المستودع كمرجع تاريخي.
