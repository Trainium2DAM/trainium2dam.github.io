# Nutrição e pratos

A seção **Pratos** é o catálogo de receitas e sugestões nutricionais da academia. Qualquer usuário pode explorar os pratos disponíveis e propor receitas próprias para adicionar ao catálogo.

---

## Explore o catálogo

Pressione **"Pratos"** na tela principal para acessar o catálogo. Os pratos são exibidos um de cada vez em formato de cartão grande, como um carrossel linear. Use as **setas de navegação** para passar para o próximo prato ou retornar ao anterior.

Somente pratos que foram **aprovados por um administrador** aparecem no catálogo. As sugestões com revisão pendente não ficam visíveis para usuários padrão.

---

## Informações sobre cada prato

Cada cartão de prato mostra:

| Dados | Descrição | 
|---|---| 
| **Foto** | Imagem do prato (se adicionada no upload) | 
| **Nome** | Nome do prato ou receita | 
| **Calorias** | Conteúdo calórico aproximado por porção | 
| **Tempo de preparação** | Tempo estimado (por exemplo, "20 minutos") | 
| **Descrição/Receita** | Ingredientes, etapas de preparação ou outras informações relevantes |

---

## Sugira um novo prato

Se quiser propor uma receita para adicionar ao catálogo, pressione o botão de sugestão (ícone de mais ou similar) e preencha o formulário:

### Campos do formulário

| Campo | Obrigatório | Descrição | 
|---|---|---| 
| **Nome do prato** | Sim | O título com que aparecerá no catálogo | 
| **Descrição/Receita** | Não | Ingredientes, passos, dicas de preparo... | 
| **Tempo de preparação** | Não | Escreva o tempo em texto livre, por exemplo “30 minutos” | 
| **Calorias** | Não | Calorias aproximadas por porção | 
| **Foto** | Não | Selecione uma imagem da sua galeria |

Ao preencher o formulário pressione **"Enviar sugestão"**.

### O que acontece a seguir?

1. Sua sugestão está registrada com status **revisão pendente**. 
2. O gerente da academia a examina. 
3. Se você aprovar, o prato aparece no catálogo visível para todos os usuários. 
4. Se você rejeitar, o prato será excluído sem que você receba notificação (verifique com o administrador se deseja saber o motivo).

:::informações
Você não precisa ser usuário Premium para sugerir pratos. Este recurso está disponível para todos os usuários registrados. 
:::

## Boas práticas na hora de sugerir pratos

Para dar à sua sugestão mais chances de ser aprovada:

- Coloque um **nome claro e descritivo** que identifique claramente o prato. 
- Inclui uma **receita detalhada** com ingredientes e etapas. Quanto mais completo for, mais útil será para outros usuários. 
- Adicione uma **foto de qualidade** do prato finalizado. Sugestões com fotos têm mais impacto visual no catálogo. 
- Indique as **calorias** caso as conheça, mesmo que sejam aproximadas. 
- Certifique-se de que o prato é **adequado ao contexto desportivo**: receitas equilibradas, fontes de proteínas, hidratos de carbono de qualidade, etc.
