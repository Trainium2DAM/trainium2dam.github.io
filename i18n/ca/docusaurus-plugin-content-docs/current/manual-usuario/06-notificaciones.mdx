# Notificacions de recordatori

Trainium pot enviar-te una notificació al mòbil per recordar-te que tens una reserva propera. Així no se t'oblida cap sessió al gimnàs.

---

## Com funcionen els recordatoris

Quan fas una reserva, l'app programa automàticament una alarma al telèfon. Quan arriba el moment configurat, reps una notificació amb:

- **Títol:** "Trainium — Recordatori de reserva" 
- **Missatge:** nom de la màquina i l'horari (per exemple, "Bicicleta estàtica: 10:00 – 10:30")

No heu de fer res per activar els recordatoris: es programen sols en confirmar cada reserva.

---

## Configurar les notificacions

Pots gestionar les notificacions des de **El meu Perfil → Edita perfil**, a la secció **"Notificacions"**. Allà trobaràs dos controls:

### Activar o desactivar les notificacions

Un **interruptor** us permet activar o desactivar els recordatoris de reserva de forma global. Si ho desactives, no rebràs cap avís encara que tinguis reserves pendents. El paràmetre es desa automàticament.

### Configurar l'antelació de l'avís

Quan les notificacions estan activades, pots triar amb quanta antelació vols rebre el recordatori prement un dels botons disponibles:

| Opció | Quan arriba l?avís | 
|---|---| 
| **10 minuts** | 10 minuts abans de l´hora d´inici | 
| **15 minuts** | 15 minuts abans | 
| **30 minuts** *(per defecte)* | 30 minuts abans | 
| **60 minuts** | 1 hora abans | 
| **120 minuts** | 2 hores abans |

El canvi es desa automàticament quan prems una opció. A partir d'aquest moment, el proper recordatori farà servir la nova configuració.

## Permís de notificacions

Per rebre els recordatoris l'app necessita permís per mostrar notificacions al telèfon.

**En Android 13 o superior:** La primera vegada que obris Trainium, el sistema et pregunta si vols permetre les notificacions. Clica **"Permet"** per activar-les.

**Si vas rebutjar el permís:** Pots activar-lo més endavant des dels paràmetres del telèfon:

1. Obre els **Configuració** del telèfon. 
2. Aneu a **Aplicacions**. 
3. Cerca i prem a **Trainium**. 
4. Entra a **Notificacions**. 
5. Activa l'interruptor.

:::info 
Si no concediu el permís de notificacions, l'app funciona amb total normalitat. Només vol dir que no rebràs els avisos de recordatori abans de les teves reserves. 
:::

---

## Reprogramació automàtica

L'app manté sempre actualitzat el recordatori de la propera reserva. Això vol dir que:

- Quan fas una **nova reserva**, s'avalua quina és la teva propera reserva futura i es programa el recordatori per a aquesta. 
- Quan **cancel·les una reserva**, el recordatori d'aquesta reserva s'elimina i es reprograma automàticament per a la següent que tinguis.

No heu de gestionar els recordatoris manualment en cap moment.

## Preguntes sobre notificacions

**Puc rebre un recordatori si el telèfon està en silenci?** 
Sí. El recordatori arriba com una notificació estàndard del sistema. Si el vostre telèfon està en silenci o en mode "No molestar", rebràs la notificació silenciosa que veuràs a la barra d'estat.

**I si el telèfon està apagat quan arriba l'avís?** 
Si el telèfon estava apagat en el moment de l'avís i l'encens després, la notificació no arribarà. Els recordatoris requereixen que el dispositiu estigui encès en el moment del tret.

**Es pot rebre el recordatori si l'app està tancada?** 
Sí. El sistema d'alarmes Android funciona encara que l'app estigui tancada en segon pla.
